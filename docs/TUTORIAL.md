# PocketClaw 使用教程

## 准备工作

你只需要准备两样东西：

1. **一个 AI 模型的 API Key**（下面教你怎么获取）
2. **网络连接**

### 获取 API Key（以 DeepSeek 为例）

1. 用浏览器打开 https://platform.deepseek.com
2. 点击"注册"，用手机号创建账号
3. 登录后，点击左边的"API Keys"
4. 点击"创建 API Key"
5. 复制生成的 Key（以 `sk-` 开头的一串字符）
6. 把它保存好，**只会显示一次**！

> 新用户有免费额度，够日常使用。

## 开始使用

### 1. 插入 U 盘

把 PocketClaw U 盘插入电脑的 USB 接口。

### 2. 双击启动

打开 U 盘，找到启动文件并双击：

- **Windows**: 双击 `启动PocketClaw.exe`
- **Mac**: 双击 `启动PocketClaw.app`

> **Mac 用户注意**：如果提示"无法打开"，请右键点击文件，选择"打开"，然后在弹窗中再点一次"打开"。这是 macOS 的安全提示，只需操作一次。

### 3. 等待浏览器打开

双击后几秒钟，浏览器会自动打开 PocketClaw 界面。

### 4. 设置 AI 模型

浏览器会自动打开配置页面：

1. 选择一个 AI 模型（推荐 DeepSeek）
2. 把刚才复制的 API Key 粘贴进去
3. 点击"开始使用"

### 5. 开始聊天

在主页点击"开始聊天"，输入你的问题，按 Enter 键发送。

## 日常操作

### 换一个 AI 模型

在主页点右上角齿轮图标 → 设置页面 → 选择新模型 → 输入对应的 API Key → 保存

### 检查更新

设置页面 → 向下滚动到"系统更新"→ 点击"检查更新"

### 使用高级功能

在主页点击"切换到高级模式"，可以使用 OpenClaw 原生管理界面，配置聊天平台（QQ、飞书、Telegram 等）、管理技能等。

### 停止使用

- **Windows**: 关闭命令提示符窗口即可
- **Mac**: 在 Dock 中右键点击 PocketClaw → 退出，或直接拔出 U 盘

### 换一台电脑

直接把 U 盘拔出来，插到另一台电脑上，双击启动即可。你的设置和聊天记录都在 U 盘里。

## 安全提示

- API Key 存储在 U 盘的配置文件中，请**妥善保管 U 盘**
- 建议在 DeepSeek 等平台设置 **月度消费限额**
- 不要在不信任的电脑上使用（如网吧）
