# 常见问题 (FAQ)

## 启动

### Q: 双击启动文件没反应？

**Windows**: 双击 `启动PocketClaw.bat`。如果弹出安全提示，点击"更多信息"→"仍要运行"。

**Mac**: 双击 `启动PocketClaw.command`。如果提示"要在 Terminal 中打开吗"，点击"打开"即可。

### Q: 启动后浏览器没有打开？

手动打开浏览器，输入地址 `http://localhost:3210` 访问。

### Q: 需要安装什么软件吗？

不需要。口袋龙虾自带运行所需的一切，双击启动即可，开箱即用。

### Q: 弹出"运行环境不完整"的错误？

说明 U 盘中的文件不完整或被损坏。请从 GitHub Releases 重新下载完整版本，或联系提供 U 盘的人获取新的拷贝。

## 使用

### Q: 支持哪些 AI 模型？

国内（无需翻墙）：MiniMax、DeepSeek、Kimi (月之暗面)、通义千问、智谱 GLM、豆包 (字节跳动)

国际（需海外网络）：Claude (Anthropic)、GPT (OpenAI)、Gemini (Google)

共 9 个服务商，每个提供 2-3 个最新模型可选。

### Q: API Key 从哪里获取？

从各模型提供商官网注册获取。推荐 MiniMax（国内首选，中文能力强）或 DeepSeek（编程首选，性价比高）。详见 [API Key 获取指南](api-keys.md)。

### Q: 要花钱吗？

口袋龙虾本身免费开源。AI 对话的费用由模型提供商收取，按用量计费。DeepSeek 等平台有免费额度，普通使用花费很少。

### Q: 可以离线使用吗？

不可以。AI 对话需要调用云端模型，必须联网。

### Q: U 盘换到别的电脑还能用吗？

可以。所有数据都在 U 盘里，拔出来插到另一台电脑上，双击启动就能继续用。

### Q: 聊天记录在哪？

在 U 盘的 `data/` 文件夹里。换电脑时会一起带走。

## 故障排除

### Q: 页面一直显示"加载中"？

等待 10-20 秒让 AI 引擎启动完成，然后刷新页面。

### Q: 发消息没有回复？

1. 检查网络连接是否正常
2. 确认 API Key 输入正确
3. 到设置页面检查模型是否选择正确
4. 重新双击启动文件试试

### Q: 如何重置所有设置？

删除 U 盘中 `data/.openclaw/openclaw.json` 文件，下次启动会自动重新引导配置。

## 聊天平台（飞书 / QQ / 微信）

### Q: 如何接入飞书 / QQ / 微信？

在引导第三步或设置页的"频道接入"Tab 中配置。详见 [频道配置指南](channels.md)。

### Q: 需要公网 IP 或服务器吗？

不需要。飞书和 QQ 使用 WebSocket，微信 ClawBot 使用长轮询，都只需要电脑能上网。

### Q: 微信 ClawBot 是什么？

腾讯官方推出的 OpenClaw 接入插件，零封号风险。需要 iOS 微信 8.0.70+，启用后在 OpenClaw 控制台扫码配对。

### Q: QQ 机器人配置后不回复？

1. 确认在 QQ 开放平台的沙箱配置中添加了自己的 QQ 号
2. 确认 App ID 和 Client Secret 填写正确
3. 检查启动窗口是否有 `[qqbot] Gateway ready` 日志

### Q: 飞书机器人配置后不回复？

1. 飞书开放平台需要创建"企业自建应用"（个人版用户需先创建团队）
2. 确认已开启机器人能力 + WebSocket 事件接收
3. 确认应用已发布
4. 需要在飞书中搜索并添加机器人到聊天

### Q: 可以同时接入多个平台吗？

可以。飞书、QQ、微信可以同时启用，互不干扰。每个平台的消息独立处理，AI 回复只发到消息来源的平台。

### Q: 想换 API Key 或切换模型怎么办？

打开 http://localhost:3210/settings，在"模型 API Key"Tab 中修改。保存后立即生效。

## 安全

### Q: 我的 API Key 安全吗？

API Key 存储在 U 盘上。请妥善保管 U 盘，建议在模型提供商处设置消费限额。

### Q: 会收集我的数据吗？

不会。口袋龙虾完全开源，所有数据存在你的 U 盘本地，不会上传到任何服务器。
