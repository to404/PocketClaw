# 常见问题 (FAQ)

## 启动

### Q: 双击启动文件没反应？

**Windows**: 如果弹出"Windows 已保护你的电脑"，点击"更多信息"→"仍要运行"。

**Mac**: 如果提示"无法打开"，右键点击 → 选择"打开"→ 在弹窗中点"打开"。

### Q: 启动后浏览器没有打开？

手动打开浏览器，输入地址 `http://localhost:3210` 访问。

### Q: 需要安装什么软件吗？

不需要。PocketClaw 自带运行所需的一切，双击启动即可，开箱即用。

### Q: 弹出"运行环境不完整"的错误？

说明 U 盘中的文件不完整或被损坏。请从 GitHub Releases 重新下载完整版本，或联系提供 U 盘的人获取新的拷贝。

## 使用

### Q: 支持哪些 AI 模型？

国内（无需翻墙）：MiniMax、DeepSeek、Kimi (月之暗面)、通义千问、智谱 GLM

国际：Claude (Anthropic)、GPT (OpenAI)

共 7 个服务商，每个提供 2-3 个最新模型可选。

### Q: API Key 从哪里获取？

从各模型提供商官网注册获取。推荐 DeepSeek (https://platform.deepseek.com)，注册就有免费额度。

### Q: 要花钱吗？

PocketClaw 本身免费开源。AI 对话的费用由模型提供商收取，按用量计费。DeepSeek 等平台有免费额度，普通使用花费很少。

### Q: 可以离线使用吗？

不可以。AI 对话需要调用云端模型，必须联网。

### Q: U 盘换到别的电脑还能用吗？

可以。所有数据都在 U 盘里，拔出来插到另一台电脑上，双击启动就能继续用。

### Q: 聊天记录在哪？

在 U 盘的 `data/` 文件夹里。换电脑时会一起带走。

## 故障排除

### Q: 页面一直显示"加载中"？

等待 10-20 秒让 AI 引擎启动完成，然后刷新页面。

### Q: 发消息没有回复？

1. 检查网络连接是否正常
2. 确认 API Key 输入正确
3. 到设置页面检查模型是否选择正确
4. 重新双击启动文件试试

### Q: 如何重置所有设置？

删除 U 盘中 `data/.openclaw/openclaw.json` 文件，下次启动会自动重新引导配置。

## 安全

### Q: 我的 API Key 安全吗？

API Key 存储在 U 盘上。请妥善保管 U 盘，建议在模型提供商处设置消费限额。

### Q: 会收集我的数据吗？

不会。PocketClaw 完全开源，所有数据存在你的 U 盘本地，不会上传到任何服务器。
